# Welcome to Metadump

Hi! This project is dedicated to a tool that let's users dump NFTs from their wallets and pass 'em further

# UPDATE as of 18th of April, 2022

Unfortunately, there is no way to run this script on a, let's say VPS, due to use of puppeteer. OpenSea (AWS to be exact) is restricting access since puppeteer (even with StealthPlugin installed) is bad at mimicking an average browser. Project is postponed until further notice.
